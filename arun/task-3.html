Introduction to Combinators

CSS combinator selectors allow you to style elements based on their relationship with other elements in the DOM tree.

Combinators are powerful tools that help you write more specific and maintainable CSS code.

The Descendant Selector

The descendant selector targets any element that is a descendant of another element.

.container p { color: blue; }

The Child Selector

The child selector targets direct children of an element.

.container > p { font-weight: bold; }

Remember that child selectors only work for direct children, not grandchildren or deeper descendants.

Adjacent Sibling Selector

This selector targets elements that immediately follow another element.

h2 + p { margin-top: 0; }

General Sibling Selector

The general sibling selector is similar to the adjacent sibling but targets all siblings that follow, not just the immediate one.

h2 ~ p { line-height: 1.8; }

Practice these selectors to become proficient in writing efficient CSS.
