CSS Combinators

Combinators explain the relationship between selectors and allow you to combine them to target specific elements.

1. Descendant Combinator (Space)

Selects all elements that are descendants of a specified element.

Syntax: ancestor descendant

Example HTML:

This paragraph is inside a div

This paragraph is inside a section, inside a div

This span is also inside the div

This paragraph is outside the div

2. Child Combinator (>)

Selects all elements that are direct children of a specified element.

Syntax: parent > child

Example HTML:

  • Direct child item 1
  • Direct child item 2
    • Nested child item 1
    • Nested child item 2
  • Direct child item 3

3. Adjacent Sibling Combinator (+)

Selects the first element that is immediately after another specific element.

Syntax: element + immediately-following-sibling

Example HTML:

Main Heading

This paragraph immediately follows the h4

This paragraph does not immediately follow the h4

This div comes after the paragraphs

4. General Sibling Combinator (~)

Selects all elements that are siblings after a specified element.

Syntax: element ~ following-siblings

Example HTML:

Reference Heading

First sibling paragraph

A div sibling

Second sibling paragraph

A span sibling

Third sibling paragraph

6. Combinator Comparison

Descendant (space)

div p

All paragraphs inside any div (any level deep)

Child (>)

div > p

Only paragraphs that are direct children of div

Adjacent (+)

h3 + p

Only the first paragraph immediately after h3

General Sibling (~)

h3 ~ p

All paragraphs that are siblings after h3

  • Descendant combinator is the most commonly used - great for styling content within containers
  • Child combinator is useful when you want to be specific about direct relationships
  • Adjacent sibling is perfect for styling elements that immediately follow others (like first paragraphs after headings)
  • General sibling is less common but useful for styling all elements of a type that come after a reference element
  • You can chain combinators: article > h2 + p selects paragraphs that immediately follow h2 elements that are direct children of articles
