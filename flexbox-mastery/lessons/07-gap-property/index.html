üìè Gap Property Types

Row gap, then column gap

üîÑ Fallback for Older Browsers

.item {
    margin: 0.5rem;
}

/* Modern browsers with gap */

    display: flex;

}

    margin: 0; /* Reset margins */
}

üíª Code Examples

/* Basic gap usage */
.container {
    display: flex;
    gap: 1rem;
}

/* Different row and column gaps */
.grid {
    display: flex;
    flex-wrap: wrap;
    gap: 2rem 1rem; /* row-gap column-gap */
}

/* Individual gap properties */
.container {
    display: flex;
    flex-wrap: wrap;
    row-gap: 2rem;
    column-gap: 1rem;
}

/* Responsive gaps */
.responsive-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

@media (min-width: 768px) {
    .responsive-grid {
        gap: 2rem;
    }
}
