Lesson 10: Flex Basis

Set the initial size of flex items before growing or shrinking

üéØ What is Flex Basis?

flex-basis sets the initial main size of a flex item before free space is distributed according to flex-grow and flex-shrink.

  • flex-basis: auto - Uses item's width/height (default)
  • flex-basis: 200px - Sets initial size to 200px
  • flex-basis: 50% - Sets initial size to 50% of container
  • flex-basis: 0 - Ignores content size, distributes space evenly
üí° Key Point: flex-basis is like width/height but specifically for the main axis!

‚ö° Flex Shorthand Property

The flex property is shorthand for flex-grow, flex-shrink, and flex-basis:

flex: 1; = flex: 1 1 0;
flex: auto; = flex: 1 1 auto;
flex: none; = flex: 0 0 auto;
flex: 2 1 200px; = grow: 2, shrink: 1, basis: 200px

Common Patterns

flex: 1
flex: auto
flex: none

‚úÖ Best Practices

‚úÖ Do: Use flex shorthand

flex: 1 1 250px;

Clear and concise

‚ö†Ô∏è Avoid: Mixing width with flex

width: 200px; flex-grow: 1;

Can cause unexpected behavior

‚úÖ Do: Use flex-basis for responsive

flex: 1 1 300px;

Items wrap when too narrow

‚ö†Ô∏è Avoid: Fixed widths in flex

width: 300px;

Not responsive

üíª Code Examples

/* Responsive card grid */
.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

.card {
    flex: 1 1 250px; /* grow, shrink, min-width */
    max-width: 350px; /* prevent too wide */
}

/* Dashboard layout */
.dashboard {
    display: flex;
    height: 100vh;
}

.sidebar {
    flex: 0 0 250px; /* fixed width sidebar */
}

.main-content {
    flex: 1 1 0; /* take remaining space */
    min-width: 0; /* allow shrinking */
}

/* Form with flexible inputs */
.form-row {
    display: flex;
    align-items: center;
    gap: 1rem;
}

.label {
    flex: 0 0 100px; /* fixed label width */
}

.input {
    flex: 1 1 0; /* flexible input */
    min-width: 150px;
}

.button {
    flex: 0 0 auto; /* size to content */
}
