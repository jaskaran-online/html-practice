Lesson 11: Align Self

Override alignment for individual flex items

üéØ What is Align Self?

align-self allows individual flex items to override the container's align-items value.

  • align-self: auto - Uses container's align-items (default)
  • align-self: flex-start - Aligns to start of cross axis
  • align-self: flex-end - Aligns to end of cross axis
  • align-self: center - Centers on cross axis
  • align-self: stretch - Stretches to fill container
  • align-self: baseline - Aligns to text baseline

üíª Code Examples

/* Featured card that stretches */
.card-container {
    display: flex;
    align-items: flex-start;
}

.card {
    flex: 1;
    padding: 1rem;
}

.card.featured {
    align-self: stretch; /* Fill full height */
    background: linear-gradient(...);
}

/* Navigation with centered logo */
.navbar {
    display: flex;
    align-items: center;
    height: 60px;
}

.logo {
    align-self: center; /* Ensure logo stays centered */
}

.nav-links {
    align-self: flex-start; /* Links at top */
}

/* Form with mixed alignments */
.form-row {
    display: flex;
    align-items: flex-start;
    gap: 1rem;
}

.label {
    align-self: center; /* Center labels */
}

.help-text {
    align-self: flex-end; /* Help text at bottom */
    font-size: 0.8rem;
    color: #666;
}

/* Icon alignment */
.content-row {
    display: flex;
    align-items: flex-start;
}

.icon {
    align-self: center; /* Center icons */
}

.text {
    align-self: flex-start; /* Text at top */
}
