Lesson 08: Flex Grow

Control how items expand to fill available space

üéØ What is Flex Grow?

flex-grow defines how much an item should grow relative to other items when there's extra space in the container.

  • flex-grow: 0 - Item won't grow (default)
  • flex-grow: 1 - Item takes equal share of extra space
  • flex-grow: 2 - Item takes twice as much space as items with flex-grow: 1

üíª Code Examples

/* Equal distribution */
.container {
    display: flex;
}

.item {
    flex-grow: 1; /* All items grow equally */
}

/* Sidebar layout */
.layout {
    display: flex;
    height: 100vh;
}

.sidebar {
    flex-grow: 0; /* Don't grow */
    width: 250px;
}

.main {
    flex-grow: 1; /* Take remaining space */
}

/* Featured item gets more space */
.card {
    flex-grow: 1;
}

.card.featured {
    flex-grow: 2; /* Twice as much space */
}
