Key Differences:

  • justify-content controls main axis (horizontal by default)
  • align-items controls cross axis (vertical by default)
  • Axes switch when flex-direction: column

üíª Code Examples

/* Perfect centering */
.card {
    display: flex;
    align-items: center;
    justify-content: center;
    height: 200px;
}

/* Form input alignment */
.input-group {
    display: flex;
    align-items: center;
    gap: 1rem;
}

/* Baseline text alignment */
.text-container {
    display: flex;
    align-items: baseline;
}

/* Stretch to fill height */
.sidebar {
    display: flex;
    flex-direction: column;
    align-items: stretch;
}
